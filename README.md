# CME295 한국어 자막 프로젝트  
# Korean Subtitles for Stanford CME295 Transformers & LLMs

Translation project of **Stanford CME295: Transformers & Large Language Models, Autumn 2025** lecture subtitles.  
Also see [CME295 Website](https://cme295.stanford.edu/) and [CME295 Cheatsheet](https://cme295.stanford.edu/cheatsheet/).

---

## Announcements

> Stanford **CME295: Transformers & LLMs** 한국어 자막 번역 프로젝트입니다.  
> 이 강의는 Transformer 구조, LLM 학습, 파인튜닝, reasoning, RAG, agent, evaluation 등 현대 LLM을 이해하는 데 필요한 핵심 내용을 다룹니다.  
> 이 프로젝트는 영어 강의 수강에 부담이 있는 한국어 학습자를 위해 영어 원본 자막을 한국어로 번역하고, 한국어 전용 자막과 한영 병기 자막을 함께 제공합니다.
 
오역, 오타, 싱크 문제는 지속적으로 수정할 예정입니다.

번역 오류나 개선 사항이 있다면 Issue 또는 Pull Request로 알려주세요.

## Usage

1. Download or clone this repository.

```bash
git clone https://github.com/chlanstjr0301/CME295_KOR_SUB.git
```

2. Open a CME295 lecture video on YouTube or download the lecture video.

3. Load the subtitle file manually in a video player.

Recommended video players:

- Windows: VLC, PotPlayer, MPC-HC
- macOS: VLC, IINA
- Linux: VLC, SMPlayer

4. Choose subtitle type.

```text
kor/          → Korean only
kor + eng/    → Korean + English bilingual
```

## Lecture Topics

### Lecture 1: Transformer

- Background on NLP and tasks
- Tokenization
- Embeddings
- Word2vec, RNN, LSTM
- Attention mechanism
- Transformer architecture

### Lecture 2: Transformer-based Models & Tricks

- Attention approximation
- MHA, MQA, GQA
- Position embeddings
- RoPE and applications
- Transformer-based architectures
- BERT and its derivatives

### Lecture 3: Large Language Models

- Definition and architecture
- Mixture of experts
- Context length, temperature
- Sampling strategies
- Prompting, in-context learning
- Chain of thought
- Self-consistency

### Lecture 4: LLM Training

- Pretraining
- Quantization
- Hardware optimization
- Supervised fine-tuning, SFT
- Parameter-efficient fine-tuning, LoRA

### Lecture 5: LLM Tuning

- Preference tuning
- RLHF overview
- Reward modeling
- RL approaches, including PPO and variants
- DPO

### Lecture 6: LLM Reasoning

- Reasoning models
- RL for reasoning
- GRPO
- Scaling

### Lecture 7: Agentic LLMs

- Retrieval-augmented generation
- Advanced RAG techniques
- Function calling
- Agents
- ReAct framework

### Lecture 8: LLM Evaluation

- LLM-as-a-judge overview
- Best practices and benefits
- Biases and pitfalls

### Lecture 9: Current Trends

- Recap
- Trending topics
- Closing thoughts
