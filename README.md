# CME295 한국어 자막 프로젝트

스탠퍼드 대학교 **CME295: Transformers & LLMs** (2025년 가을 학기) 강의의 한국어 자막 번역 프로젝트입니다.

## 강좌 소개

- **강좌명**: Stanford CME295 — Transformers & Large Language Models
- **학기**: Autumn 2025
- **언어**: 영어 원본 → 한국어 번역

## 폴더 구조

```
CME295_KOR_SUB/
├── eng/          # 영어 원본 자막 (.srt)
├── kor/          # 한국어 번역 자막 (.srt)
└── kor + eng/    # 한영 병기 자막 (.srt)
```

## 사용 방법

- `kor/` 폴더의 자막: 한국어 전용으로 시청할 때 사용
- `kor + eng/` 폴더의 자막: 원문과 번역을 함께 볼 때 사용 (학습 추천)

자막 파일을 영상과 같은 폴더에 놓거나, 미디어 플레이어(VLC, MPC-HC 등)에서 자막을 수동으로 불러오세요.
